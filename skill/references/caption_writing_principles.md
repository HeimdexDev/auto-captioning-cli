# Caption Writing Principles — Spatiotemporal Flow (Step 2)

캡션은 이 데이터셋의 **메인 산출물**이다. 목적은 장면의 **시공간 흐름**(시간에 따른 동작·위치·구도의 변화)을 설명하는 것. 메타 필드는 캡션 완성 후 채운다.

## 원칙 1. 30~80자, 한 문장 기본

- 너무 짧으면 흐름 정보가 안 들어가고, 너무 길면 평가 메트릭이 불안정.
- 시간 전개가 둘로 뚜렷이 나뉘면 두 문장 허용 (예: "먼저 ~한다. 이어서 ~로 바뀐다").
- 자수는 Unicode 글자수(공백 포함).

## 원칙 2. 흐름을 드러내라 — 주체 + 변화 + 순서

좋은 흐름 캡션은 **무엇이**(주체/대상) + **어떻게 변하는지**(동작·이동·구도 변화) + **순서**(먼저→이어서→끝)를 담는다.

| 요소 | 예시 |
| --- | --- |
| 주체/대상 | 여성 1명 / 두 사람 / 손 / 화면 속 자막 카드 |
| 변화 | 이동한다 / 돌아선다 / 들어 올린다 / 카메라가 줌인된다 / 좌→우로 팬한다 / 장면이 전환된다 |
| 순서 | 먼저 ~, 이어서 ~, 끝에 ~ / ~로 시작해 ~로 끝난다 |

정적 씬이면 변화 대신 **공간 구성**을 묘사하고 "거의 변화 없다"를 명시한다.

## 원칙 3. 시공간 그라운딩 — 보이는 변화만, 추측 금지

- ❌ "방을 가로질러 걸어간다" (2장 사이의 경로는 알 수 없음 — 날조)
- ❌ "고급스럽게 회전한다" (평가어)
- ❌ "샤넬 신상을 소개한다" (음성·브랜드 추측)
- ✅ "왼쪽에 서 있던 여성이 다음 프레임에서 중앙으로 이동해 정면을 본다" (프레임 간 실제 변화)
- ✅ "세 프레임 모두 구도가 거의 같고 인물의 손 위치만 조금 바뀐다" (변화가 작을 때 정직하게)

화면에 보이는 자막/브랜드명은 적어도 되지만 `text_on_screen` 필드가 우선. 캡션에서는 "하단 자막이 바뀐다"처럼 시각 사실로.

## 원칙 4. 표기 사전

- 인물: "여성 1명", "남성 1명", "두 사람", "여성 쇼호스트"(명확할 때), 부분 신체 "손"/"발". 이름·평가어 금지.
- 색상: 빨강·분홍·노랑·주황·초록·파랑·보라·검정·흰 (외래어는 대체어 없을 때만: 베이지·카키·네이비·와인·아이보리·브라운).
- 변화·이동 동사 권장: 이동하다, 돌아서다, 다가가다, 물러나다, 들어 올리다, 내려놓다, 펼치다, 가리키다, 회전하다, 전환되다, (카메라) 줌인/줌아웃하다, 패닝하다, 컷이 바뀌다.
- 정적 상태 동사: 놓여 있다, 서 있다, 걸려 있다, 표시되어 있다.

## 원칙 5. 3개 ref는 서로 다른 시간적 렌즈

같은 씬에 3개 캡션이 모이므로 표현이 같으면 multi-reference 데이터셋 가치가 없다(BLEU/METEOR/CIDEr는 다양한 ref 일치를 본다).

| ref | 렌즈 | 무엇을 강조 |
| --- | --- | --- |
| `ref_a` | **행동·사건 흐름** | 주체가 프레임을 거치며 하는 동작의 순서 |
| `ref_b` | **공간·시점 변화** | 위치·구도·카메라가 프레임마다 어떻게 바뀌는지 |
| `ref_c` | **장면 요약 서사** | 시작→끝을 한 문장으로 압축 |

### 다양성 = 어순 변경 ❌, 시간적 시점 변경 ✅

같은 씬 (여성이 왼쪽에서 중앙으로 이동하며 제품을 들어 올리고, 카메라가 살짝 줌인):

❌ 어순만:
- ref_a: "여성이 중앙으로 이동하며 제품을 들어 올리고 카메라가 줌인된다"
- ref_b: "카메라가 줌인되는 가운데 여성이 제품을 들어 올리며 중앙으로 이동한다"

✅ 렌즈가 다름:
- ref_a (행동): "여성이 먼저 제품을 집어 든 뒤 이어서 카메라를 향해 들어 올려 보여준다"
- ref_b (공간·시점): "왼쪽에 있던 인물이 프레임을 거치며 중앙으로 옮겨지고 화면이 조금씩 줌인된다"
- ref_c (요약): "여성이 제품을 들어 보이는 동작으로 시작해 클로즈업으로 마무리되는 장면"

## 셀프 체크

- [ ] 30~80자인가
- [ ] 시간 변화(동작/이동/구도) 또는 정적이면 "변화 없음"이 드러나는가
- [ ] 보이지 않는 중간 동작을 날조하지 않았는가
- [ ] 색상·표기 사전을 지켰는가
- [ ] 3개 ref가 행동 / 공간·카메라 / 요약 서사로 시점이 다른가
